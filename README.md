# 거리 측정 앱

이 웹앱은 OpenStreetMap Nominatim API를 이용해 출발지와 도착지를 조회하고 두 지점 간 총 거리를 계산합니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 출발지와 도착지를 입력합니다.
3. `거리 계산` 버튼을 클릭하면 거리와 지도 보기 링크가 표시됩니다.

## Netlify 배포

1. 이 폴더를 Netlify에 업로드하거나 Git 리포지토리로 연결합니다.
2. `Publish directory`를 `.`로 설정합니다.

앱이 배포되면 정적 HTML/CSS/JavaScript로 서비스됩니다.
